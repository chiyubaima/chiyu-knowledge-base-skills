# Chiyu Knowledge Base Skills

Local Codex skills for maintaining Chiyu's Obsidian knowledge base at:

`/Users/chiyu/Documents/chiyu-knowledge-base`

## Skills

- `wiki-ingest`: compile source material into maintained wiki pages.
- `wiki-query`: answer questions using only local wiki pages.
- `wiki-lint`: inspect and repair knowledge base health issues.
- `wiki-spark`: quickly capture ideas and excerpts into the raw layer.

These skills preserve a two-layer model: raw source records stay under `raw/`, while maintained knowledge lives under `wiki/`.
